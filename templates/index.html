Описание работы с веб-каталогом реестров прогнозных ресурсов

Загрузка новых документов

  1. Для загрузки новых реестров формата (xls, xlsx) нужно выбрать вкладку верхнего меню «Загрузка новых реестров».
  2. В появившейся форме загрузки нужно выбрать файл реестра для загрузки. Название файла должно содержать только латинские буквы, в названии недопустимы пробелы и любые другие символы кроме «_».
  3. В текстовой поле нужно ввести название реестра, название должно соответствовать субъекту РФ для которого составлен реестр.
  4. Нажать на кнопку «Загрузить».
  5. При выполнении загрузки могут быть выведены ошибки, которые необходимо исправить и снова загрузить файл.

Скачивание загруженных реестров

  1. Для скачивания загруженного реестра нужно выбрать вкладку верхнего меню «Скачать реестр».
  2. Выбрать из списка нужный реестр и нажать на его название.
  3. Для актуализации загруженного реестра нужно установить пункт «Скачать для актуализации».

Актуализация загруженных реестров

  1. Скачать реестр для актуализации.
  2. Внести изменения в соответствии с технологической инструкцией «Ведение реестров.....».
  3. В скаченном реестре для актуализации есть служебные поля: «filename, id_reg, _id, _rev». Данные поля необходимы для корректной загрузки актуализированного реестра, их редактирования запрещено.
  4. Для загрузки реестров необходимо выбрать вкладку верхнего меню «Загрузка актуализированных реестров».
  5. В появившейся форме загрузки нужно выбрать файл актуализированного реестра для загрузки. Название файла должно содержать только латинские буквы, в названии недопустимы пробелы и любые другие символы кроме «_».
  6. В выпадающем списке выбрать реестр для актуализации.
  7. Нажать кнопку «Загрузить».
  8. При выполнении загрузки могут быть выведены ошибки, которые необходимо исправить и снова загрузить файл.
